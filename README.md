# Exam-seating auditor

---

📘 Suggested README Structure

1. Project Title & Description
- **Title**: *Exam Seating Auditor*
- **Description**: A tool to automate and manage exam seating arrangements for students, ensuring fairness, efficiency, and easy auditing.

2. Features
- Generate seating plans automatically.
- Prevent students from sitting next to classmates with the same subject.
- Export seating charts to Excel/PDF.
- Admin dashboard for managing exams and rooms.

 3. Installation
```bash
# Clone the repository
git clone https://github.com/your-username/Exam-seating.git

# Navigate into the project folder
cd Exam-seating

# Install dependencies
pip install -r requirements.txt
```


4. Usage
```bash
python main.py
```
- Input exam details (date, subject, student list).
- Generate seating arrangement.
- Export or print seating chart.
